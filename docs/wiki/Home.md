# Needle-X Wiki

Needle-X is a local-first runtime that turns noisy web pages into compact, proof-carrying context for AI agents.

This wiki is the shortest navigation layer for the project.

Start here if you want:
1. a quick orientation

## What Needle-X Is

1. a compact web context compiler
2. a proof-carrying retrieval layer
3. a local-first runtime with CLI and MCP surfaces

## What Needle-X Is Not

1. a browser agent
2. a generic search engine
3. a raw scraper with no provenance model
